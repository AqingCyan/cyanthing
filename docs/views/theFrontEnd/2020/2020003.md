## 确定技术栈是否稳健

首先，我们需要总结2019年前端届发生了哪一些变化，技术的新趋势是什么？自己的技术栈是否足够稳健。这里有一篇很不错的文章可以参考：[倏忽一年：回顾点评前端 2019 重大事件、趋势](https://www.infoq.cn/article/xd8DNvVZ28zBc846mqJE)

从文章中可以看到：

- <span style="color: #61dafb">**React**</span> 与 <span style="color: #31af7c">**Vue**</span> 在这一年依然火热，虽然 <span style="color: #cc2b39">**Angular**</span> 和 <span style="color: #31af7c">**Vue**</span> 持平，但鉴于国内厂家使用较少，还是推荐 <span style="color: #61dafb">**React**</span> 与 <span style="color: #31af7c">**Vue**</span> 作为自己的首选框架。但这里面的会发现 **jQuery** 十分显眼，但结合大环境，很明显，现在它已经不适合学习了。
- <span style="color: #337ab7">**TypeScript**</span> 使用率激增，在 2019 年初发布的 StackOverflow 调查中，<span style="color: #337ab7">**TypeScript**</span> 与 **Python** 并列最受欢迎语言次席，仅次于 **Rust**。所以2019年到现在，<span style="color: #337ab7">**TypeScript**</span> 再次增长，也不会奇怪。
- <span style="color: #61dafb">**React Hooks**</span> 变得十分受欢迎。作为2018年末发布的新特性，2019年我们可以在很多的技术性平台看到大家对它的讨论，并且都是正面相关的内容，<span style="color: #61dafb">**React**</span> 开发人员已经离不开它了，那么将 <span style="color: #61dafb">**React**</span> 作为首选的同学，一定需要掌握好它的知识点。
- <span style="color: #31af7c">**Vue 3.0**</span>预备发版，那么在春招的时候，也不可避免会被面试官考察到你是否掌握了一些比较前沿的技术，以此来观察你的学习能力。
- 等等.....

可以明确的感受到，其实变化与趋势虽然很强烈，但其实主要还是再框架方面的改变，以及前端开发工程相关的部。如果你的技术栈是从围绕以上提到的内容，本质需要我们在复习的时候去加深了解和准备的其实并没有发生太大的变化，我们只需要在框架和工程方面有一些侧重点即可。
